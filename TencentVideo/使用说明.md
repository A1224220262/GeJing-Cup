# 使用说明

#### 需要的第三方库

```
selenium
PIL
numpy
opencv-python
```

#### 运行命令

```
python main.py
```

#### 运行结果

```
程序文件如下：
spider.py        # 爬虫代码
main.py          # 使用数据，拼接小熊的代码
bear.jpg         # 小熊的模板
运行之后会生成的目录或文件如下：
data/data.json   # 爬虫爬取的数据
images/*.jpg     # 下载的图片
portrait.jpg     # 用户头像
result.png       # 拼接成的小熊图片
```
